# MD→Word 实验报告批量转换

将指定目录下的 Markdown 实验报告文件批量转换为 Word (.docx) 格式。

## 输入

$ARGUMENTS — 包含 .md 文件的目录路径（如未提供，使用当前工作目录）

## 工作流程

1. **扫描目录**：查找所有 `*.md` 文件（递归子目录）
2. **检查依赖**：确认 `python-docx` 已安装，未安装则 `pip install python-docx`
3. **生成转换脚本**：写入临时 Python 脚本 `md2docx.py`，包含以下功能：
   - 解析 Markdown 标题层级（`#` → Heading 1, `##` → Heading 2 等）
   - 处理加粗、斜体、代码块、表格、列表
   - 设置中文字体（宋体/微软雅黑）和标准字号
   - 页面设置：A4 纸，上下 2.54cm，左右 3.18cm 页边距
   - 表格样式：带边框，表头加粗
   - 代码块：等宽字体，灰色背景
4. **批量执行**：对每个 .md 文件生成同名 .docx
5. **清理**：删除临时脚本

## 输出格式

- 每个 `.md` 文件生成同目录下的同名 `.docx`
- 报告转换结果：成功/失败计数

## 注意事项

- 中文路径需要 UTF-8 编码处理
- Windows PowerShell 环境下使用 `python`（非 `python3`）
- 图片引用保留（相对路径），不存在的图片跳过
