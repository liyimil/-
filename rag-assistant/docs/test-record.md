## CLI 手动验证

### 资料内问题

命令：

```bash
python src/main.py "Day1要交什么？"
```

预期：

- 输出应提到 Day1 提交物或 6 类证据文件。
- 输出末尾应包含来源编号。
- 来源应至少包含 `[faq-02]` 或其他与 Spec / 交付相关的 FAQ。

实际：

- 输出包含 `spec`、`design`、`ai-log`、`6 类证据文件` 等内容。
- 输出包含来源：`[faq-04]`, `[faq-02]`, `[faq-01]`。

### 资料外问题

命令：

```bash
python src/main.py "奖学金政策是什么？"
```

预期：

- 不回答奖学金政策。
- 输出 `资料中没有找到依据`。

实际：

```text
资料中没有找到依据。本助手仅能回答 Day1 AI Native 训练营相关问题。
```
