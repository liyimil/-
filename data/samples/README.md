# samples 目录说明

用于存放手工构造的输入样例或企业脱敏样本说明。

当前样例：

```text
samples-md/
  alarms.json
  features.json
  rules.json
```

该样例包含告警采样数据、特征定义数据和事件规则数据。

---

## 告警字段说明（`alarms.json`）

### 原始字段

| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| `alarm_id` | string | 是 | 告警唯一标识，格式如 `<前缀>-<编号>` |
| `station_name` | string | 是 | 厂站名称（原始字段，可能为空或占位符） |
| `device_name` | string | 否 | 设备名称（原始字段，当前数据中均为空字符串） |
| `signal_name` | string | 是 | **核心字段**，包含告警全部业务信息的自然语言文本 |
| `signal_value` | int | 否 | 信号值，`1`=动作/产生，`0`=复归/消失 |
| `voltage_level` | string | 否 | 电压等级（原始字段，当前数据中均为空字符串） |
| `alarm_time` | string | 否 | 告警时间（原始字段，仅 CSV 测试数据中有，可为空） |
| `alarm_level` | string | 否 | 告警级别（原始字段，仅 CSV 测试数据中有，可为空） |

### `signal_name` 格式分析

告警数据包含两种格式：真实脱敏数据和 CSV 测试占位数据。

**真实格式**，结构为 `{时间} {厂站} {设备对象} {动作状态}`：

```text
{YYYY}年{MM}月{DD}日 {HH}:{MM}:{SS} {XX}kV{某变电站} {某线路}保护A网中断 动作(全数据判定)
{YYYY}年{MM}月{DD}日 {HH}:{MM}:{SS} {XX}kV{某变电站} {某开关} 分闸
{YYYY}年{MM}月{DD}日 {HH}:{MM}:{SS} {XX}kV{某变电站} {某刀闸} 分闸
{YYYY}年{MM}月{DD}日 {HH}:{MM}:{SS} {XX}kV{某变电站} {某开关} 合闸
{YYYY}年{MM}月{DD}日 {HH}:{MM}:{SS} {XX}kV{某变电站} {某线路}测控A网中断 动作
```

其中：
- **时间**：`YYYY年MM月DD日 HH:MM:SS`
- **厂站**：`[电压等级]变电站名`，如 `XXkV某变电站`
- **设备对象**：`开关`、`刀闸`、`接地刀闸`、`线路保护`、`线路测控` 等，可能带编号前缀
- **动作状态**：`分闸` / `合闸` / `动作` / `复归` / `A网中断` 等，可能带后缀如 `(全数据判定)`

**测试占位格式**：`测试信号N`，不包含可解析的结构化信息。
